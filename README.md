# 虎尾科技大學校園 AI 助手

校園設施地圖系統與 AI 助手，提供校園內廁所、飲水機、垃圾桶等設施的位置查詢和智能導航。

## 功能特色

- 🗺️ 校園設施地圖查詢
- 🤖 AI 智能助手對話
- 📍 GPS 導航功能
- 📊 設施狀態管理
- 🌐 中英文雙語支援

## 技術棧

- **前端**: HTML, CSS, JavaScript, Leaflet
- **後端**: Rasa (Python)
- **部署**: Vercel (前端), Railway (Rasa)
